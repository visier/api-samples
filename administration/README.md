# Administration

Samples that use Visier Administration APIs for [Visier software development lifecycle (SDLC)](https://docs.visier.com/developer/Embedded/get-started/sdlc.htm#SDLC) workflows across development, validation, and production environments.

## Samples

* [SDLC promotion sample](sdlc-promotion/python) shows a guarded Python workflow for exporting committed project changes from a source environment and importing them into a draft project in a target environment, with publishing available only when explicitly requested.
